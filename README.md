# X (Twitter) UI Clone

A responsive UI clone of **X (formerly Twitter)** built with **HTML** and **Tailwind CSS**. This project focuses on recreating the platform's interface while practicing responsive layouts, Flexbox, CSS Grid, sticky positioning, and modern UI design techniques.

## ✨ Features

* Responsive layout for mobile and desktop
* Mobile bottom navigation
* Desktop sidebar navigation
* Sticky right sidebar
* Search bar
* "What's Happening" section
* "Who to Follow" section
* Interactive hover and focus states
* Material Symbols icons
* Hidden scrollbar for sidebar while preserving scroll functionality

## 🛠️ Built With

* HTML5
* Tailwind CSS
* Google Material Symbols
